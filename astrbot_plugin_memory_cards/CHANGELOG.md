# 更新日志

## v0.3.0

- 新增 schema v3 与 note revisions，更新和合并会保留变更历史
- 新增本地候选召回、结构化 create/update/merge/noop 解析和候选 ID 白名单校验
- 自动整理改为事务型应用，非法模型输出或越权 ID 不落库并按原机制重试
- 新增模型判定的质量整理预览、稳定指纹、过期拒绝、确认应用和历史 API
- WebUI 新增质量整理入口、整理前后对照、确认/取消面板和变更历史视图

## v0.2.0

- 新增私聊用户与机器人回复的持久化短期缓冲
- 新增消息数或空闲时间任一满足时的静默自动整理
- 新增当前会话模型与可选专用整理模型支持
- 新增严格 JSON 校验、凭据过滤、完全重复去重和失败退避
- 后台增加自动生成标识与待整理消息数量
- 数据库自动从 schema v1 原地迁移到 v2，不清空已有便签

## v0.1.0

- 新增私聊用户隔离的 SQLite 便签存储
- 新增后台便签卡片页面和管理员 CRUD 接口
- 新增本地相关性检索与本轮临时 LLM 上下文注入
